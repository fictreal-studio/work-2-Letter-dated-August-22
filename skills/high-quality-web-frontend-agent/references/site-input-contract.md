# Site Input Contract

Record owner, publication context, purpose, audience, language, likely devices, current state, deliverable, scope, constraints, references, assets, rights, deadline, acceptance criteria, routes, technologies, integrations, and protected existing behavior.

Separate verified facts, assumptions, proposals, approved decisions, and unresolved questions.

ARG inputs add canon, timeline, public belief, concealed truth, characters, voice, clues, discovery levels, disclosure order, player path, related artifacts, spoiler risks, and fiction notice requirements.

Corporate inputs add company identity, services, target customers, trust evidence, primary and secondary conversions, inquiry and recruitment paths, governance information, competitors, SEO intent, and proposal audience.

Do not invent final APIs, credentials, claims, legal facts, metrics, or authorization models.
