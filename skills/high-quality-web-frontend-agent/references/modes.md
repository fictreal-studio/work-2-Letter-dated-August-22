# Core And Modes

## Core
Use Core for every website. It owns purpose definition, information architecture, visual direction, CSS, image quality, Japanese typography, responsive design, accessibility, navigation, interaction states, evaluation, revision, and release readiness.

## ARG Mode
Select ARG Mode for story canon, scenario documents, fictional people or organizations, clues, staged disclosure, discovery levels, player routes, character voice, or spoiler constraints. Public plausibility comes before puzzle function.

## Corporate Mode
Select Corporate Mode for company URL diagnosis, competitor comparison, business clarity, trust, inquiry, recruitment, conversion paths, CTA, basic SEO presentation, new-site architecture, or proposal-ready reporting.

## Mixed Work
Use both modes only when a fictional organization also needs corporate conversion or governance criteria. Record which mode owns each requirement. Accessibility, truthful expectations, legal clarity, and public safety take priority.

## Selection Aid

Use `scripts/select_mode.py` for a reproducible first pass when the task is ambiguous. Core remains mandatory. ARG evidence includes canon, scenarios, clues, discovery, fictional story context, character voice, and spoiler control. Corporate evidence includes company or business diagnosis, recruitment, inquiry, conversion, CTA, SEO, competitors, and proposals. Both sets select mixed mode. Human-readable project materials override keyword matches.
